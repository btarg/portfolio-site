Project Overview

A Portal-inspired first-person puzzle game built in Unity using C#.

The game's puzzles revolve around freezing objects and rewinding them in time, similar to the mechanics later seen in Nintendo's The Legend of Zelda: Tears of The Kingdom.

This project was developed as a personal learning project and later used as my final assignment for my Computer Science A-Level course in May of 2022, a year before Tears was released.

Key Features

  • Time Manipulation: Freeze objects and rewind them to previous positions
  • Progressive Difficulty: Three carefully designed levels with increasing complexity
  • Save/Load System: Complete game state persistence between sessions
  • Physics Integration: Seamless interaction between time mechanics and Unity's physics
  • Visual Feedback: Clear indicators for frozen objects and time states, as well as post-processing shaders for effect
