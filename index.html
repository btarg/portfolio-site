A Portal - inspired puzzle game built in Unity using C#.
The game's puzzles revolve around freezing objects and rewinding them in time, similar to the mechanics later seen in Nintendo's The Legend of Zelda: Tears of The Kingdom.
The game has three levels, a main menu, and save-load functionality.
This project was started as a personal testbed for learning Unity around 2019, several years before Tears was released.
I used this as my final assignment for my Computer Science A-Level in May 2022.

A Minecraft mod which allows a streamer's Twitch chat to control their game, using a custom command system.
The mod was built using Minecraft Forge in Java.
It was designed to be the easiest to use and most flexible chat control mod available, with a custom command system which allows for easy expansion by other mod developers.
It is available from CurseForge and Modrinth, and has been downloaded over 20,000 times.
I started making this mod in 2019 while I was still at school, and it has been used by several popular streamers. I turned some of their highlights into my own YouTube videos.
