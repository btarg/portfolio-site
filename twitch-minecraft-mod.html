Project Overview

A popular Minecraft Forge mod that revolutionizes streaming by allowing Twitch chat to directly control the streamer's game experience, built with Java and the Twitch API. The mod has over 21,000 downloads across CurseForge and Modrinth platforms, and has been played by several popular streamers.

The initial goal for this project was to provide an easy-to-use mod with minimal setup for streamers, with pre-set commands designed to be fun and intuitive for viewers to use, such as spawning an anvil above the player, or randomising their inventory.

I started building Twitch Vs Minecraft in 2019, when I was still in school, using it as a playground to hone my programming skills until 2022 by providing consistent updates with new content and support for later versions of Minecraft.


Key Features

  • Custom Command System: Flexible framework for creating and managing chat commands
  • Developer-Friendly API: Extensible system allowing other mod developers to add functionality
  • Configurable Controls: Streamers can customise which commands are available and tweak cooldowns
